6. Stage 2: GPU Tokenization Loop

build_cutset() Load shar_index.json → CutSet.from_shar() → round-robin split across ranks → filter (sr, duration)
↓
DynamicBucketingSampler Group clips by duration into 20 buckets → fill batches up to max_batch_duration (2000s)
↓
DataLoader 32 workers decode audio from Shar • prefetch_factor=4 • pin_memory
↓
handler.process_batch() Audio → GPU → WavTokenizer → audio tokens • Read text_tokens from cut.custom • Assemble & write
↓
Checkpoint (every 500 batches) Finalize chunk → atomic rename .tmp → save sampler state + stats → open next chunk

7. Multi-Node Architecture

Fault-isolated: no NCCL, no collectives. Each rank is fully independent.

Shard assignment: shards[rank::world_size] — each rank reads/writes independently
No NCCL
No collectives, no deadlocks. One rank crash doesn't affect others.
Per-Rank Checkpoint
Each rank saves rank_XXXX_checkpoint.pt independently. Resume restores sampler state.
Atomic Writes
Write to .tmp, then os.replace(). No partial files on crash.

11. Quick Reference

# Step 1: Prepare Shar (includes text tokenization)
python -m audio_tokenization.utils.prepare_data.prepare_wds_to_shar \
    --wds-shards '/data/shards/*.tar' \
    --shar-dir /output/shar \
    --text-tokenizer /path/to/tokenizer.json \
    --target-sr 24000 --num-workers 32

# Step 2: GPU Tokenize (direct)
CUDA_VISIBLE_DEVICES=0,1,2,3 torchrun --nproc_per_node=4 \
    -m audio_tokenization.tokenize \
    dataset=audio_text/emilia_yodas_isolated \
    mode=audio_text audio_text_format=direct audio_text_task=transcribe

# Step 2: GPU Tokenize (interleaved)
CUDA_VISIBLE_DEVICES=0,1,2,3 torchrun --nproc_per_node=4 \
    -m audio_tokenization.tokenize \
    dataset=audio_text/emilia_yodas_interleaved \
    mode=audio_text audio_text_format=interleaved

# Verify: decode a sample back to audio
python -c "
import torch, torchaudio
from audio_tokenization.vokenizers.wavtokenizer.audio_only import WavTokenizerAudioOnly
from audio_tokenization.utils.indexed_dataset import MMapIndexedDataset
tok = WavTokenizerAudioOnly('/path/to/omni_tokenizer', device='cuda')
ds = MMapIndexedDataset('/path/to/rank_0000_chunk_0000')
_, audio, info = tok.detokenize(torch.tensor(ds[0]).long().cuda())
torchaudio.save('decoded.wav', audio.unsqueeze(0).cpu(), info['output_sample_rate'])
"
